Principais Formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag <b></b> (não semântico).

Nesta frase temos um termo em destaque usando a tag <strong></strong> (semântico).

Itálico / Enfase

Nesta frase temos um termo em itálico usando a tag <i></i> (não semântico).

Nesta frase temos um termo em enfase usando a tag <em></em> (semântico).

Texto marcado

Podemos também criar um termo marcado usando a tag <mark></mark> .

e no outro paragrafo temos um novo termo marcado usando a tag <mark></mark> com outra cor aplicada .

Texto grande e texto pequeno

Nesta frase temos um texto grande usando a tag <big></big> (não semântico).

Nesta frase temos um texto pequeno usando a tag <small></small> (semântico).

Texto deletado

Podemos marcar um texto como excluido usando a tag <del></del> .

Texto inserido

Podemos marcar um texto como inserido usando a tag <ins></ins> (semântico).

Existe também o texto sublinhado usando a tag <u></u> (não semântico).
